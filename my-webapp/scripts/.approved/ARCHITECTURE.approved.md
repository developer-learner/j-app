# Frozen Architecture Contract

> This file is a **snapshot** of `docs/ARCHITECTURE.md` at the time the PRD was approved.
> The test agent reads **this** file, not the live `docs/ARCHITECTURE.md`, so a mid-loop
> re-plan cannot change the contract under running tests.

## How to use

1. When a PRD is approved, the PM copies `docs/ARCHITECTURE.md` here:
   ```bash
   cp docs/ARCHITECTURE.md scripts/.approved/ARCHITECTURE.approved.md
   ```
2. The test agent reads this file — it is the sole API/design contract.
3. If the architect re-plans mid-loop, `docs/ARCHITECTURE.md` changes but this
   snapshot stays frozen until the next PRD approval.
